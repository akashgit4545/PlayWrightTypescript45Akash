# PlayWrightTypescript45Akash

Setup Playwright TypeScript Project

Check system requirements - https://playwright.dev/docs/intro#system-requirements

Step 1 - Check if node.js is installed node -v npm -v Step 2 - If not Install Node.JS. Can download zip folder and add path to PATH env variable

Step 3 - Install IDE (VS Code)

Step 4 - Create a folder and open in VS Code

Step 5 - Open terminal on the project folder and run command ---> npm init -y

Step 6 - Check package.json file is created 

Step 7 - Run command ----> npm init playwright@latest   & select TypeScript when asked

This will: Install Playwright Setup TypeScript config Create playwright.config.ts, tests/, and other helpful files

Step 8 - Run the default test that comes with Playwright: npx playwright test
